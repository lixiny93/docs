# 快速开始

在Github仓库中打开Release
[Github仓库](https://github.com/lixiny93/SB-Chat)

选择最新版本的Release文件，只有1个Release文件和2个源代码文件，你不会分不清吧

下载后解压，在运行前请确认你安装了nodejs及npm环境

然后

<Tabs groupId="operating-systems">
  <TabItem value="win" label="Windows">运行 start.bat</TabItem>
  <TabItem value="mac" label="其他系统">
    运行命令

    node index.js


  </TabItem>
</Tabs>

然后就没有然后了
